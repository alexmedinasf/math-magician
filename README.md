# Math Magicians

> Calculator App using React.


## Built With

- JavaScript
- React
- CSS

## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Modern Web Browser
- NPM updated
- Git

### Install

    npm install
### Usage

    npm run build
    npm start
